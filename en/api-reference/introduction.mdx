<Note>
  La API de MetaQare permite a hospitales, aseguradoras, clínicas y desarrolladores integrarse fácilmente con nuestro ecosistema, automatizar procesos y acceder a información clave de manera segura.
</Note>

## ¿Qué puedes hacer con la API de MetaQare?

- Crear, leer y actualizar información de pacientes, citas y profesionales.
- Gestionar autorizaciones, planes de salud privados y para aseguradoras.
- Automatizar flujos administrativos y de atención médica.
- Integrar tus propios sistemas, aplicaciones o portales externos.
- Acceso seguro y controlado bajo estándares internacionales de privacidad.

<Card title="Autenticación" icon="leaf" href="https://github.com/mintlify/starter/blob/main/api-reference/openapi.json">
  La autenticación de la API se realiza mediante **API keys** o tokens de acceso seguros.

  Solicita tu acceso escribiendo a [**soporte@metaqare.ai**](mailto:soporte@metaqare.ai), si aún no tienes tus credenciales.
</Card>

## Ejemplo de solicitud

Una consulta básica a la API usando `curl`:

```bash
curl -X GET "https://api.metaqare.ai/pacientes" \
  -H "Authorization: Bearer TU_API_KEY"
